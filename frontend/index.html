使用说明

一、准备 API 服务

这个工具需要调用 AI 来答题，你需要一个能用的 AI 接口。

推荐：Kilo.ai（免费，开箱即用）
  1. 打开 kilo.ai 注册账号
  2. 登录后，点击左侧 "API Keys"
  3. 点 "Create API Key"，复制密钥（类似 ki_xxxxxxxx）
  4. 回到工具，填入配置：
    • 接口地址：https://api.kilo.ai/api/openrouter
    • 接口密钥：粘贴刚才复制的密钥
    • 模型名称：点击「选择模型」选一个带 free 的免费模型

其他兼容 OpenAI 格式的 API 也可以用（如 DeepSeek、OpenAI 官方等）。

二、代理配置（可选）

如果网络无法直接访问 AI 服务，可以填代理地址（如 http://127.0.0.1:7890），不需要就留空。

三、基本使用

1
填写考试信息

在 examcoo.com 打开考试页面，复制浏览器地址栏的完整链接，填入姓名、工号、部门。

2
选择执行模式

自动答题：抓取题目、AI 回答、自动提交、查成绩。
扫题入库：只抓取题目保存到本地，不调用 AI，不提交。

3
点击「开始执行」

底部日志区会实时显示进度，等它跑完就行。想中途停止点「停止」按钮。

四、参数说明

参数说明建议值
投票轮数同一道题让 AI 答多次，取出现最多的答案。轮数越多越准，但也越慢越费钱。3（默认）
答题轮次程序自动提交多少次。设为多轮会循环执行「抓题-答题-提交-查分」。1（默认）
请求延迟每次 AI 请求之间的等待时间，避免请求过快。1 秒
答题时间每轮从开始答题到提交的最短时间，模拟真人答题节奏。3 秒

五、题库管理

程序自动维护本地题库 answer_bank.json（在 exe 同目录）。

  • 查看题库：可按题型、状态、关键词筛选
  • 编辑题库：点击题目修改答案，可标记为已校验
  • 自动更新：答对的题自动校验，答错的题会修正或删除旧答案

已校验的题目下次直接使用，不再调用 AI，既快又省钱。

六、支持的题型

  • 单选题（A/B/C/D 选一个）
  • 多选题（A/B/C/D 选多个）
  • 判断题（正确/错误）

填空题、简答题等不支持，会自动跳过。

七、常见问题

Q：显示 "API Key 无效"
检查接口密钥是否填对，有没有多余空格，用「测试连通」按钮验证。

Q：显示 "模型不存在"
模型名称写错了，点「选择模型」先拉取列表再选。

Q：考试链接怎么获取？
在浏览器打开考试页面（能看到题目或开始按钮），复制地址栏完整 URL。

Q：程序一直在跑没结果？
可能是 AI 接口响应慢，看日志区有没有红色错误。投票轮数设得高需要等久一点。
